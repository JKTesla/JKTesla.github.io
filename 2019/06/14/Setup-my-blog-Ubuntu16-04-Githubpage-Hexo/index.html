Setup my blog : Ubuntu16.04 + Githubpage + Hexo

Jun 14, 2019

Keyword：Hexo， 文件结构

前言：建立个人博客是由来已久的想法。我是个表达欲比较强的人，无论是想法还是情绪。但觉得想法、情绪只当落在纸面时，推敲过，才更有深度和价值（不排除说来道去的是满纸荒唐言），辟一块空间时时自省。

加之，网友友情提示：You should have a personal web site、6 reasons why you should blog as a software developer，也是我写作的动力。

简介

Github用户可申请Githubpage免费域名用于挂载静态网页，所以我们精力可放在静态网页配置及博文撰写。

我所知的静态网页生成工具有：Jekyll（Ruby编写）、Hexo（JavaScript编写）、Hugo（Go编写），工具的benchmark可参考：靜態網站產生器大比拚。

我的博客使用Hexo创建，原因有二：

  • Plugins插件很多：Hexo官网-Plugins
  • 部署到Githubpage，操作简便：hexo-deployer-git

Hexo静态网页搭建：

1、配置指南

参考：Hexo Documentation

解决方案：查看版本并升级 yarn、Node.js（推荐ppa升级）、core-js

2、文件结构 ⭐⭐⭐⭐⭐

熟悉Hexo文件及theme文件结构：我的博客搭建过程 ，测试Hexo插件部署及站内搜索功能添加。

3、theme选配

本博客采用的theme是 tranquilpeak，操作对象主要是：Hexo项目目录下的_config.yml、source/_posts下的md文件、theme目录下的_config.yml，更鼓励开发自己的theme。模板theme可在 Hexo官网-Themes 浏览选配。
